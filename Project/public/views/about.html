This site was developed by Gilad Shimony & Shay Talyas from Ben-Gurion University of the Negev, Department of Software & Information Systems Engineering.

At the beginning of the work we thought deeply about the responsibilities of the server side and the client side.

We mapped all the functions to be implemented on the server and client side. For example, the products should be sorted on the client side.

In addition, client programming at Angular was very challenging and required a lot of investment from us.

We encountered some difficulties during our work.

For example: syntax issues and compilation errors - it is very difficult to debug the code and sometimes it takes a very long time to understand where the mistake is.

In addition, we spent a lot of time to fully understand the functionality and all its parts before we started programming.

In conclusion, we can say that we learned a lot from this assignment and even enjoyed it during work :)
